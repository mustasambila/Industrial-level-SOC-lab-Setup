# MISP to Wazuh Integration

## Overview

MISP provides threat intelligence enrichment for alerts generated within Wazuh.

## Objectives

* IOC Enrichment
* Threat Intelligence Correlation
* Threat Actor Context
* Campaign Tracking

## Indicators Imported

* Malicious IP Addresses
* Domains
* URLs
* File Hashes
* Email Indicators

## Workflow

Threat Feed

→ MISP

→ Wazuh

→ Alert Correlation

## Security Benefits

* Improved alert accuracy
* Enhanced investigations
* Faster threat identification
* Reduced analyst workload

## Outcome

Threat intelligence successfully enriches Wazuh detections.
