# TheHive to Wazuh Integration

## Overview

TheHive provides incident management and case tracking for alerts generated by Wazuh.

## Objectives

* Incident Tracking
* Alert Management
* Case Documentation
* Response Coordination

## Workflow

Wazuh Alert

→ Case Creation

→ Analyst Investigation

→ Incident Resolution

## Incident Lifecycle

1. Detection
2. Validation
3. Investigation
4. Containment
5. Eradication
6. Recovery
7. Lessons Learned

## Benefits

* Structured incident response
* Case documentation
* Analyst collaboration
* Audit trail

## Outcome

Wazuh alerts are transformed into actionable incidents within TheHive.
