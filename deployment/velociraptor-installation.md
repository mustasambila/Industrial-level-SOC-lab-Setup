# Velociraptor DFIR Deployment Guide

## Overview

Velociraptor is deployed to provide Digital Forensics and Incident Response (DFIR) capabilities across enterprise endpoints.

## Objectives

* Endpoint Visibility
* Threat Hunting
* Incident Response
* Artifact Collection
* Malware Investigation

## Server Configuration

Ubuntu Server 22.04

## Deployment Architecture

Server:
192.168.10.20

Agents:

* Windows 11
* Ubuntu Linux

## Capabilities

* Live Response
* Process Analysis
* Registry Analysis
* Event Collection
* Memory Investigation

## Use Cases

* Persistence Detection
* Malware Investigation
* Insider Threat Analysis
* Lateral Movement Detection

## Integration

Integrated with:

* Wazuh
* Sysmon
* TheHive

## Outcome

Velociraptor successfully deployed as the DFIR platform for the SOC laboratory.
