# pfSense Firewall Deployment Guide

## Overview

pfSense is deployed as the perimeter firewall of the SOC laboratory. It provides routing, firewall filtering, NAT, VPN capabilities, network segmentation, and acts as the primary network security control between the internet and internal systems.

## Objectives

* Deploy a perimeter firewall
* Create network segmentation
* Enable secure traffic flow
* Support IDS/IPS deployment
* Centralize network logging

## VM Configuration

| Setting           | Value      |
| ----------------- | ---------- |
| OS                | pfSense CE |
| RAM               | 4 GB       |
| CPU               | 2 vCPU     |
| Storage           | 20 GB      |
| Network Adapter 1 | WAN        |
| Network Adapter 2 | LAN        |

## Installation Procedure

1. Download the latest pfSense ISO.
2. Create a virtual machine in VMware Workstation.
3. Attach the ISO image.
4. Allocate resources.
5. Boot the VM.
6. Select Install pfSense.
7. Accept default filesystem settings.
8. Complete installation and reboot.
