# Threat Hunting Playbook: Suspicious PowerShell Activity

## Hunting Objective

Identify malicious or suspicious PowerShell execution on Windows systems.

## Threat Hypothesis

An attacker may be abusing PowerShell for execution, persistence, privilege escalation, or malware delivery.

## MITRE ATT&CK Mapping

Technique:
T1059.001

Tactic:
Execution

## Data Sources

* Sysmon Event ID 1
* Wazuh
* Windows Event Logs

## Investigation Process

### Step 1

Search for:

powershell.exe

### Step 2

Review command line arguments.

Look for:

* EncodedCommand
* DownloadString
* Invoke-WebRequest
* Invoke-Expression
* Base64 Encoded Content

### Step 3

Identify parent processes.

Examples:

* cmd.exe
* winword.exe
* excel.exe
* explorer.exe

### Step 4

Review network connections generated by PowerShell.

### Step 5

Identify downloaded files and persistence mechanisms.

## Indicators of Suspicious Activity

* Encoded PowerShell commands
* Hidden PowerShell windows
* Network downloads
* Child process creation

## Escalation Criteria

Escalate if PowerShell:

* Downloads files
* Contacts unknown IPs
* Creates persistence
* Executes encoded commands

## Expected Outcome

Identify PowerShell abuse before successful compromise occurs.
