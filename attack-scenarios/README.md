# Attack Scenarios

## Overview

This section documents security attack simulations conducted within the Enterprise SOC Home Lab.

The objective is to validate detection capabilities, test alert generation, improve incident response procedures, and evaluate security controls.

Each scenario includes:

* Attack Objective
* Attack Methodology
* Detection Sources
* MITRE ATT&CK Mapping
* Security Findings
* Lessons Learned

## Detection Technologies

* Wazuh SIEM
* Suricata IDS/IPS
* Zeek Network Security Monitoring
* Sysmon
* Velociraptor

## Simulated Threat Categories

* Reconnaissance
* Credential Attacks
* Malware Activity
* Persistence
* Lateral Movement
* Command and Control
* Data Exfiltration

## Outcome

These simulations validate the effectiveness of the SOC architecture and improve analyst readiness for real-world incidents.
