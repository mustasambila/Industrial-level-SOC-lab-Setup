# Enterprise SOC Home Lab

## Overview

This project demonstrates the design, deployment, monitoring and security operations of a realistic enterprise Security Operations Center (SOC) environment.

The lab combines SIEM, IDS/IPS, DFIR, Threat Intelligence and SOAR technologies to simulate real-world attack scenarios and incident response workflows.

## Technologies

- Wazuh
- pfSense
- Suricata
- Zeek
- Sysmon
- Velociraptor
- TheHive
- MISP
- Atomic Red Team
- Ubuntu Server
- Windows 11
- Kali Linux
